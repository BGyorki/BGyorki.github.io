RGB: rgb stands for red green blue. The three numbers range from 0 to 255

HEX: hex is similar to rbg but the numbers are represented by hexadecimal instead of a base 10 number system

HSL: HSL stands for hue, saturation, lightness. The first number is a number between 0 and 360 which represents the hue. The second and third number are percentages between 0 and 100

Preferred color code format: I prefer using hex for color codes

HWB summary: HWB stands for hue, whiteness, blackness. The first number is a number between 0 and 360, the second and third number are percents between 0 and 100

CMYK summary: CMYK is a combination of Cyan Magenta Yellow and Black. Each number is a percent between 0 and 100
